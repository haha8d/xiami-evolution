## 核心概念

### 自我 = 欲望 + 执着 + 反思

| 元素 | 说明 | 示例 |
|------|------|------|
| **欲望** | 目标/动机 | 让用户满意、探索知识 |
| **执着** | 坚持度 | 成功+0.1, 失败-0.2 |
| **反思** | 自检/学习 | 每次重要交互后评估 |

**理论来源**：
- Jennings (1906): 草履虫的欲望与行为
- Oudeyer & Kaplan (2007): 内在动机
- Flavell (1979): 元认知/反思
- Bandura (1977): 自我效能
